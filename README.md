# devifylabs (minimal React scaffold)

This is a minimal React + Vite scaffold placed directly in the project root. Theme: black & white, minimal.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```
